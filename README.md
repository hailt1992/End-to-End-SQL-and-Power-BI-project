Analyze the Global Unemployment
1991 - 2025


This project utilizes historical employment data (1991–2025) to perform an in-depth diagnostic of the global labor market. By implementing complex SQL queries to calculate Unemployment Recovery Rates following the 2020 economic shock, the study highlights the pace of stabilization across different regions. Additionally, the report features a comparative analysis of the gender unemployment gap, visualized through Power BI to identify long-term trends in labor equity.
